Project Name: Task Manager App

Features:
Task Management:

Users can add, edit, delete, and mark tasks as completed.
Task prioritization (high, medium, low) with different visual indicators.
Due date setting for tasks.
Categories/Tags:

Ability to group tasks by category or tag (e.g., Work, Personal, etc.).
Search and Filter:

Search tasks by name or filter by completion status or priority level.
Local Storage:

Save tasks in the browser's local storage so data is persisted even after refresh.
